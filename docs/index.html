ビデオ起動ツール「再生くん」

ビデオ起動ツール「再生くん」は、ビデオを使ったグループ活動の振り返り（リフレクション）やビデオ分析を支援するための様々な機能を提供することを目指して開発中のツールです。 最重要の機能の一つとして、Microsoft Excelを使って時間情報のリストを作っておけば、その時間情報からビデオを再生できる機能を備えています。

ご利用にあたって

「再生くん」には現在Windows版とMac版があります。

それぞれの動作環境として、いくつかのソフトウェアが事前にインストールされていることを想定しています。ご利用のPCの環境をご確認のうえ、足りないものがあればマニュアルを見ながらインストールしておいて下さい。

本ページからは、

  • 「再生くん」本体（マニュアル同梱）
  • 「再生くん」マニュアルだけ
  • 「LimeChat」利用のための補足資料（リアルタイムでインデックスを作る方法）

の3種類をダウンロードすることができます。それぞれWindows版とMac版がありますので、必要なものを選んでください。

 

Windows版をご利用の方はこちらから

Windows版の動作環境

  1. Microsoft Windows（7以上推奨、XPでも動作確認済み）
  2. Microsoft .NET Framework（4以上推奨、無料で入手可能）
  3. Microsoft Excel（2003以上推奨）

※.NET FrameworkはWindows 7に標準搭載されているため、新たにインストールしなくても「再生くん」を起動できることがあります。「再生くん」が起動できない場合には、 .NET Frameworkが入っていない（またはバージョンが古い）可能性を疑ってみて下さい。
